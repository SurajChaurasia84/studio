# **App Name**: AttendPlus

## Core Features:

- Phone Number Login with OTP: Allow users to log in using their phone number and a one-time password (OTP) for secure authentication.
- Profile Setup: Enable new users to set up their profile by providing their name, choosing a profile photo, and specifying their position (teacher/student).
- Home Screen with Quick Actions: Display a home screen with the user's name and position, a notification icon, profile photo, and quick actions such as adding leave or accessing communities. Also feature an analytics dashboard for student and class details.
- Attendance Marking: Show class name, real-time clock, date, day, and timestamps with side-by-side scroll views. Include options to change classes and mark attendance (present/absent) with alphabetized student list (A-Z), profile picture, name, and roll number.
- Class Management: Display all classes of the user like WhatsApp chats, view class details with student info (profile pic, name, roll number), and share class links with barcodes (expiring in 3 days) for students to join the class.
- Report Generation: Include an analysis bar to see user, student, attendance and class details. Feature to download class or student attendance reports as PDF.
- Settings Management: User profile and details displayed with options to change settings related to the application and the ability to logout.

## Style Guidelines:

- Primary color: Blue (#3498db) to evoke trust and professionalism, useful for an educational platform.
- Background color: Light gray (#ecf0f1), almost white to provide a clean and modern interface.
- Accent color: Orange (#e67e22) used for interactive elements and call to actions. It provides sufficient contrast, being complementary to blue.
- Font pairing: 'Poppins' (sans-serif) for headlines and 'PT Sans' (sans-serif) for body text, offering a blend of modern clarity and readability.
- Use consistent and clear icons from a set like Material Design Icons, with themed icons that indicate function in each portion of the application.
- Employ a tabbed interface for main navigation (Home, Attendance, Classes, Report, Settings), ensuring a clear and intuitive user journey.
- Use subtle transitions and animations for feedback on interactions, such as marking attendance or navigating between sections, to enhance user engagement.
